# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

We are committed to providing a welcoming, inclusive, and respectful environment for everyone, regardless of background, identity, or experience level.

Unacceptable behavior includes harassment, discrimination, intimidation, abusive language, or any other behavior that creates a hostile environment.

Maintainers are responsible for enforcing this code and will take appropriate action, including warnings or removal from the project, for violations.

We encourage constructive feedback, collaboration, and open communication in all discussions, issues, and pull requests.

For any concerns or reports of violations, please contact the maintainers directly. All reports will be treated confidentially and respectfully.
